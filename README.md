# Planejador de Obras

Este repositório contém um esquema inicial para um planejador de obras, que poderá ajudar a gerir e acompanhar tarefas, prazos e recursos em um canteiro de obras.

## Estrutura do repositório

- `openapi/` – contém especificações OpenAPI para integração com serviços externos. Actualmente inclui `clickup_openapi.json`, uma especificação da API do ClickUp usada para integrar tarefas e projetos do ClickUp ao planejador.
- `README.md` – este documento, com informações gerais sobre o projeto e instruções de contribuição.

## Como contribuir

Sinta-se à vontade para abrir issues com sugestões ou pull requests com melhorias. Todos os comentários são bem-vindos.
